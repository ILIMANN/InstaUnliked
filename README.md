A lightweight, background-running Python automation script that uses Playwright to remove likes from your Instagram activity log.

**Disclaimer:** Automating actions on Instagram violates their Terms of Service. This script is for educational purposes. Use it at your own risk. The script includes randomized delays to mimic human behavior, but aggressive use may still result in action blocks or account bans.

## How to use?

- Firstly, run the setup file to install required libraries.
- Run the *save_cookies.py* file and enter your Instagram account. This will save your log in cookie *(in a .json file)* and save time. Follow the further instructions that show up in your terminal. Later on run the *unliker_bot.py* and watch the magic happen!

## Features

- **Headless Execution:** Runs silently in the background without taking over your mouse or screen. If you want to see the code in action use the showaction code.

## Prerequisites

- Python 3.x installed with pip
